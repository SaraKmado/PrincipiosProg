INSTALAÇÃO DO DATA.LIST.SPLIT
1. Aceder ao site https://hackage.haskell.org/package/split
2. Na secção Downloads do site, ir buscar o 1o link (split-0.2.3.3.tar.gz)
3. Exportar a pasta split-0.2.3.3 do .tar.gz transferido
4. Abrir a linha de comandos
5. Ir à pasta dos downloads
6. Correr o comando cabal update
7. Correr o comando cabal install split-0.2.3.3
